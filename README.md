# Datapot Power BI Practice

Repository tổng hợp tài liệu thực hành Power BI và Data Analytics của Datapot.

## Cấu trúc thư mục

| Thư mục | Mục đích |
| --- | --- |
| `learning-paths/` | Lộ trình học có cấu trúc (PL-300, visualization). |
| `event-materials/` | Tài liệu webinar/workshop/sự kiện cộng đồng. |
| `showcase-demos/` | Demo báo cáo độc lập dùng cho blog/showcase. |
| `challenges/` | Đề bài, rubric, tài liệu cho challenge. |
| `shared-resources/` | Tài nguyên chia sẻ và dataset dùng chung. |

## Quy ước tổ chức

1. Đặt file theo nhóm nội dung trước, rồi theo series/sự kiện.
2. Dùng tên thư mục và file nhất quán, ưu tiên dạng dễ đọc và ổn định.
3. Với file trùng nội dung (hash giống nhau), chỉ giữ 1 bản chính trong repo.
