# Datapot Power BI Practice

Kho tài nguyên thực hành Power BI và Data Analytics của Datapot, được tổ chức theo nhóm nội dung để dễ tìm kiếm, tái sử dụng và bảo trì.

## Mục tiêu repository

- Lưu trữ tài liệu học tập, demo report, challenge và tài nguyên sự kiện.
- Chuẩn hóa cấu trúc thư mục để team/community dễ đóng góp.
- Giảm trùng lặp file và giữ một nguồn chuẩn cho mỗi nội dung.

## Cấu trúc chính

| Thư mục | Nội dung |
| --- | --- |
| `learning-paths/` | Lộ trình học có cấu trúc (PL-300, visualization, lab packs). |
| `event-materials/` | Tài liệu webinar/workshop/sự kiện cộng đồng Datapot. |
| `showcase-demos/` | Các demo `.pbix` độc lập cho blog và showcase. |
| `challenges/` | Đề bài, rubric và tài nguyên challenge. |
| `shared-resources/` | Tài nguyên dùng chung: study plan, career pack, tài liệu theo domain. |

## Trạng thái branch

Nội dung từ các nhánh trước đây đã được rà soát và hợp nhất vào `main`:
- `Datapot-Webinar`
- `Tài-nguyên-Data-Analytics-and-Business-Intelligence`
- `Chia-sẻ-tài-liệu-cộng-đồng`

## Quy ước quản lý nội dung

1. Ưu tiên phân loại theo **nhóm nội dung** trước, sau đó theo **series/sự kiện**.
2. Tên file/thư mục cần rõ nghĩa, ổn định và nhất quán ngôn ngữ trong cùng một nhánh nội dung.
3. Không giữ nhiều bản giống hệt nhau; file trùng nội dung cần được deduplicate.
4. File dung lượng lớn nên kèm `README.md` mô tả nguồn và mục đích sử dụng.
